## Understanding LLM Search Engines

### How LLM Search Engines Work
1. **Crawling**: LLMs crawl and index web content
2. **Semantic Understanding**: They understand context and meaning
3. **Answer Generation**: They generate answers from indexed content
4. **Citation**: They cite sources when providing answers

### Key LLM Search Engines
- **ChatGPT (OpenAI)**: Uses GPTBot for crawling
- **Claude (Anthropic)**: Uses anthropic-ai and Claude-Web crawlers
- **Gemini (Google)**: Uses Google-Extended crawler
- **Perplexity**: Uses PerplexityBot crawler
- **Microsoft Copilot**: Uses Bingbot

### What LLMs Look For
1. **Structured Data**: JSON-LD schemas
2. **Clear Answers**: Direct, concise answers to questions
3. **Authoritative Content**: High-quality, well-sourced content
4. **Semantic Clarity**: Clear context and meaning
5. **FAQ Format**: Question-answer format
6. **Comprehensive Coverage**: Complete information on topics
